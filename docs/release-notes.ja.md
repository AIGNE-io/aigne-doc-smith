# リリースノート

AIGNE DocSmith の最新の開発状況を常に把握できます。このページでは、ツールの各バージョンにおける新機能、パフォーマンス改善、バグ修正の履歴を時系列で提供します。

## バージョン 0.9.0-beta (2025-11-11)

これは、全般的な安定性とパフォーマンスの向上に焦点を当てたメンテナンスリリースです。

## バージョン 0.8.16-beta.1 (2025-11-11)

### 新機能

- **ドキュメントアイコン生成**: ドキュメントのアイコンを自動的に生成する機能を追加しました。
- **評価レポートの自動オープン**: 生成後に評価レポートを自動的に開くように設定できるようになりました。

### バグ修正

- **プロンプトの改善**: 内部プロンプトを微調整し、より表現力豊かで詳細なドキュメントを生成するようにしました。

## バージョン 0.8.16-beta (2025-11-10)

### 新機能

- **チャットアシスタントの強化**: 対話型チャット Agent が AIGNE File System (AFS) を利用してプロジェクトのコンテキストにアクセスするようになり、より正確で関連性の高い支援を提供できるようになりました。
- **タイトルの効率化**: ドキュメントのタイトルを自動的に効率化し、標準化する機能を実装しました。

### バグ修正

- **モデルの更新**: チャットモデルをカスタムハブから公式の `google/gemini-2.5-pro` に更新し、パフォーマンスを向上させました。

## バージョン 0.8.15 (2025-11-07)

このバージョンには、全般的なメンテナンスと安定性の向上が含まれています。

## バージョン 0.8.15-beta.16 (2025-11-07)

### 新機能

- **カスタマイズ可能なダイアグラム生成**: ドキュメント用に生成されるダイアグラムの数を指定できるようになりました。

## バージョン 0.8.15-beta.15 (2025-11-07)

### バグ修正

- **設定システム**: AI の「思考努力」を管理するための 2 層の設定システムを追加し、生成品質と速度をより細かく制御できるようになりました。

## バージョン 0.8.15-beta.14 (2025-11-06)

### 新機能

- **推論努力の設定**: AI モデルの推論努力を調整する設定を導入し、分析の深さをカスタマイズして制御できるようになりました。
- **公開認証の強化**: 公開プロセスでショートリンクの応答と、認証問題に関するより詳細なエラー報告を提供するようになりました。

### バグ修正

- **設定の互換性**: 既存の設定が互換性を維持するためにデフォルトの推論努力を使用するようにしました。

## バージョン 0.8.15-beta.13 (2025-11-05)

### バグ修正

- **公開の安定性**: 有料デプロイメントプロセスの安定性とスムーズさを改善しました。
- **検証と表示**: 初期化時にドキュメントディレクトリを検証し、公開後に最新バージョンが表示されるようにしました。

## バージョン 0.8.15-beta.12 (2025-11-05)
