本文档总结了该工具每个版本的新功能、改进和错误修复。所有变更均按时间倒序排列。

## 版本 0.8.12-beta.3 (2025-10-09)

### 错误修复

- 解决了发布到云端时错误地要求管理员权限的问题。

- 优化了文档生成和更新流程，以提高可靠性。
