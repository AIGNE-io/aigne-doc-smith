# 更新文件

隨著專案的發展，您的文件也需要跟上腳步。本節提供了一個中心樞紐，用於處理所有與修改文件集相關的任務，確保其保持準確和相關。在這裡，您將找到關於更改現有內容、整合新文件和移除過時檔案的指引。

保持文件最新是一個持續的過程。無論您是根據使用者回饋來完善說明、為新功能新增指南，還是透過移除過時頁面來重組內容，所提供的工具都能幫助您有效管理這些變更。

## 總結

本節是您在文件初次建立後，管理其生命週期的入口。透過遵循連結的指南，您可以隨著專案的成長和變化，自信地更新、新增或移除文件檔案。

關於從頭開始建立一個全新文件集的完整演練，請參閱 [建立文件](./guides-generating-documentation.md) 指南。
