# Terminal Command Playback

## Init & Generate Docs

```bash
aigne doc create
```

Terminal playback data: `generate.json`

## Update Docs

```bash
aigne doc update
```

Terminal playback data: `update.json`

## Translate Docs

```bash
aigne doc localize
```

Terminal playback data: `translate.json`

## Publish Docs

```bash
aigne doc publish
```

Terminal playback data: `publish.json`
